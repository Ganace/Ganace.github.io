## CSS3的calc()如何使用

---

{% highlight ruby %}

.bgdiv {
	width: 500px;
	background: #26a69a;
	padding: 5px;
}
.calc {
	background:  #26a69a;
	padding: 15px;
	border: 5px solid green;
	height: 50px;
	width: 90%;/*写给不支持calc()的浏览器*/
	width:-moz-calc(100% - (15px + 5px) * 2);
	width:-webkit-calc(100% - (15px + 5px) * 2);
	width: calc(100% - (15px + 5px) * 2);
}
{% endhighlight %}
